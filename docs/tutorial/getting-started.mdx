## Overview

Welcome to the Farrow documentation!

Farrow(Functional and arrow, pronounced `/ˈferō/`) is a Type-Friendly Web Framework for Node.js. It consists of three major parts:

* Type-Friendly HTTP Server.
* Type-Friendly and End-to-end code generational RPC Server.
* Full stack famework for a web project.

If you have questions about anything related to Farrow, you're always welcome to ask our community on [GitHub Discussions](https://github.com/farrow-js/farrow/discussions).

#### System Requirements

* [Node.js 14.18.1](https://nodejs.org) or later
* [TypeScript 4.3.0](https://www.typescriptlang.org) or higher（Can be installed with scaffolding if farrow is not introduced in an existing project）
* MacOS, Windows (including WSL), and Linux are supported

## Setup

Init project by creating a empty Node.js project.

<Tabs groupId="package-tool">
<TabItem value="npm" label="npm">

```bash
mkdir farrow-project
cd ./farrow-project
npm init -y
```

</TabItem>
<TabItem value="yarn" label="Yarn" default>

```bash
mkdir farrow-project
cd ./farrow-project
yarn init -y
```

</TabItem>
</Tabs>

Install **Farrow** libraries

<Tabs groupId="package-tool">
<TabItem value="npm" label="npm">

```bash
npm install farrow-http
```

</TabItem>
<TabItem value="yarn" label="Yarn" default>

```bash
yarn add farrow-http
```

</TabItem>
</Tabs>

Create `src/index.ts` as entry and add the following content

```ts title="src/index.ts"
import { Http, Response } from 'farrow-http'

const http = Http()

http.use(() => {
  return Response.text('Hello Farrow')
})

http.listen(3000, () => {
  console.log('server started at http://localhost:3000')
})
```

Install development tools

<Tabs groupId="package-tool">
<TabItem value="npm" label="npm">

```bash
npm install --dev farrow typescript
```

</TabItem>
<TabItem value="yarn" label="Yarn" default>

```bash
yarn add -D farrow typescript
```

</TabItem>
</Tabs>

Add `scripts` in `package.json`

```json title="package.json"
{
  "scripts": {
    "dev": "farrow dev",
    "build": "farrow build",
    "start": "farrow start"
  }
}
```

Start server with:

<Tabs groupId="package-tool">
<TabItem value="npm" label="npm">

```bash
npm run dev
```

</TabItem>
<TabItem value="yarn" label="Yarn" default>

```bash
yarn run dev
```

</TabItem>
</Tabs>

Open your browser and navigate to <http://localhost:3000/>.

## Setup with Template

We also recommend creating a new Farrow app using `create-farrow-app`, which sets up everything automatically for you. To create a project, run:

<Tabs groupId="package-tool">
<TabItem value="npm" label="npm">

```bash
npm init farrow-app farrow-project
```

</TabItem>
<TabItem value="yarn" label="Yarn" default>

```bash
yarn create farrow-app farrow-project
```

</TabItem>
</Tabs>

Select a template.

```
? Select a template:
❯ farrow-only
  farrow-vite-react
  farrow-next-react
```

After the initialization is complete, follow the instructions to start the development server.

<Tabs groupId="package-tool">
<TabItem value="npm" label="npm">

```bash
cd farrow-project
npm install
npm run dev
```

</TabItem>
<TabItem value="yarn" label="Yarn" default>

```bash
cd farrow-project
yarn
yarn dev
```

</TabItem>
</Tabs>

Open Farrow Playground and connect to <http://localhost:3000/api/todo>.

:::info
For more information on how to use `create-farrow-app`, you can review the [`create-farrow-app` documentation](/docs/api/framework/create-farrow-app).
:::
