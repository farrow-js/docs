Tool for Transforming [farrow-schema](./01-farrow-schema.mdx) to [json schema](https://json-schema.org/).

## Installation

<Tabs groupId="package-tool">
<TabItem value="npm" label="npm">

```bash
npm install farrow-json-schema
```

</TabItem>
<TabItem value="yarn" label="Yarn" default>

```bash
yarn add farrow-json-schema
```

</TabItem>
</Tabs>

## Usage

```ts
import { Int, Type } from 'farrow-schema'

const definition = {
  [Type]: Int,
}

const jsonSchema = transform(definition)
/**
    {
      $id: '/farrow/schema',
      $ref: '#/definitions/0',
      definitions: {
        0: {
          $id: '0',
          type: 'integer',
        },
      },
    }
 ** /
```

## Api

**Type Signature:**

```ts
const transform: (input: FieldDescriptor) => JSONSchema7;
```
